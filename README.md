## 有哪些轮子

* 已完成轮子：Button、Tabs、Dialog、Switch

1. 安装依赖
    ```
    yarn install
    ```

2. 启动 dev server
    ```
    yarn serve / npm start
    ```

3. 一键构建和部署
    ```
    sh ./deploy.sh
    ```
